# サイト構成・画面遷移設計書

**作成日**: 2026-04-14
**バージョン**: 1.0

---

## 1. サイトマップ

```
/ (トップページ)
├── /about (事業概要)
│   └── #profile (代表者紹介セクション)
├── /services (サービス)
├── /contact (お問い合わせ)
├── /not-found (404エラーページ)
├── /sitemap.xml (自動生成)
└── /robots.txt (自動生成)
```

## 2. ページ一覧・セクション構成

### 2.1 トップページ (`/`)

| 順序 | セクション | コンポーネント | 概要 |
|------|-----------|---------------|------|
| 1 | ヒーロー | `HeroSection` | メインビジュアル + キャッチコピー + バッジ |
| 2 | 事業概要 | `AboutSection` | 事業紹介の要約 + 「詳しく見る」リンク |
| 3 | サービス紹介 | `ServicesSection` | 3サービスカードの要約 + 「詳しく見る」リンク |
| 4 | CTA | `CtaSection` | お問い合わせ誘導 |

### 2.2 事業概要ページ (`/about`)

| 順序 | セクション | コンポーネント | 概要 |
|------|-----------|---------------|------|
| 1 | ページヘッダー | `PageHeader` | ページタイトル |
| 2 | 事業概要 | `AboutDetail` | 事業説明 + 対象者リスト |
| 3 | 事務所情報 | `OfficeInfo` | 事務所情報テーブル |
| 4 | 代表者紹介 | `ProfileSection` | 代表者プロフィール + 資格情報 |
| 5 | CTA | `CtaSection` | お問い合わせ誘導 |

### 2.3 サービスページ (`/services`)

| 順序 | セクション | コンポーネント | 概要 |
|------|-----------|---------------|------|
| 1 | ページヘッダー | `PageHeader` | ページタイトル |
| 2 | サービス一覧 | `ServicesDetail` | 3サービスの詳細説明 |
| 3 | CTA | `CtaSection` | お問い合わせ誘導 |

### 2.4 お問い合わせページ (`/contact`)

| 順序 | セクション | コンポーネント | 概要 |
|------|-----------|---------------|------|
| 1 | ページヘッダー | `PageHeader` | ページタイトル |
| 2 | フォーム + 連絡先 | `ContactForm` + `ContactInfo` | 2カラム構成 |

### 2.5 404ページ (`/not-found`)

| 順序 | セクション | コンポーネント | 概要 |
|------|-----------|---------------|------|
| 1 | エラーメッセージ | - | 404メッセージ + トップへ戻るボタン |

## 3. 画面遷移図

```
┌─────────────────────────────────────────────────┐
│                    Header (共通)                  │
│  [ロゴ → /] [事業について → /about]               │
│  [サービス → /services] [代表者 → /about#profile] │
│  [お問い合わせ → /contact]                        │
└─────────────────────────────────────────────────┘

トップ (/)
  │
  ├── 「事業について詳しく見る」 ──→ /about
  ├── 「サービスを詳しく見る」 ──→ /services
  ├── 「お問い合わせ」(CTA) ───→ /contact
  │
  ▼
/about ←──────────────────────────────────────┐
  │                                            │
  ├── 「お問い合わせ」(CTA) ───→ /contact     │
  │                                            │
/services ←────────────────────────────────────┤
  │                                            │
  ├── 「お問い合わせ」(CTA) ───→ /contact     │
  │                                            │
/contact                                       │
  │                                            │
  └── フォーム送信完了 → サクセスメッセージ表示  │
                                               │
/not-found                                     │
  └── 「トップへ戻る」 ──→ / ─────────────────┘

┌─────────────────────────────────────────────────┐
│                    Footer (共通)                  │
│  [営業時間 9:00〜18:00 平日／土日祝日・年末年始休み]│
│  [ロゴ]                    [© コピーライト]       │
└─────────────────────────────────────────────────┘
```
