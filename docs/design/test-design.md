## 4. テスト実行コマンド

```bash
# ユニットテスト実行
npm run test

# ユニットテスト（ウォッチモード）
npm run test:watch

# E2Eテスト実行（ローカル）
npx playwright test

# E2Eテスト実行（本番URL）
BASE_URL=https://xxx.vercel.app npx playwright test

# E2Eテスト（UIモード）
npx playwright test --ui

# 特定テストファイルのみ
npx playwright test tests/e2e/contact.spec.ts
```
