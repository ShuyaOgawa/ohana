## 1. プロジェクト概要

### 1.1 背景・目的

ソーシャルサポートOhanaは、千葉県船橋市を拠点とする独立型社会福祉士事務所である。専門職後見人として、認知症・知的障害・精神障害のある方の生活と権利をサポートする事業を行っている。

本プロジェクトでは、事業内容・サービス・代表者情報を発信し、相談・問い合わせの窓口として機能するコーポレートサイトを構築する。

### 1.2 サイトのゴール

- 事業内容とサービスをわかりやすく伝える
- 専門職後見人・独立型社会福祉士としての信頼感を与える
- 相談・問い合わせへのハードルを下げる
- 家庭裁判所・専門機関からの後見人候補者検索に対応する

## 2. ターゲットユーザー

| ターゲット | 詳細 |
|-----------|------|
| **一次ターゲット** | 認知症・知的障害・精神障害のあるご本人のご家族 |
| **二次ターゲット** | 成年後見制度の利用を検討している方 |
| **三次ターゲット** | 後見人候補者をお探しの専門機関・家庭裁判所 |
| **四次ターゲット** | 福祉サービスの利用について相談したい方 |

### 2.1 ユーザー特性

- 年齢層はやや高め（40代〜70代が中心と想定）
- ITリテラシーは低〜中程度
- スマートフォンからのアクセスが多いと想定
- 信頼性・安心感を重視する

## 5. デザイン要件

### 5.1 デザインコンセプト

上品で落ち着いた雰囲気。福祉分野の信頼感と温かみを両立する。

**キーワード**: 信頼、温かさ、上品、落ち着き、専門性

### 5.2 カラースキーム

アーティファクトのカラーパレットを踏襲する。

| 変数名 | カラーコード | 用途 |
|--------|-------------|------|
| `--cream` | `#f9f5ee` | セクション背景（交互） |
| `--warm-white` | `#fdfaf5` | メイン背景 |
| `--green-deep` | `#2d4a3e` | メインカラー（ヘッダー、ヒーロー背景、CTA） |
| `--green-mid` | `#4a7c6b` | セクションラベル、アクセント |
| `--green-light` | `#7aab94` | サブアクセント |
| `--green-pale` | `#c8ddd6` | テキスト（暗い背景上） |
| `--sand` | `#c9b99a` | 装飾線、ボーダー |
| `--sand-light` | `#e8dccb` | 薄い装飾 |
| `--text-dark` | `#1e2e28` | 本文テキスト（主要） |
| `--text-mid` | `#4a5e57` | 本文テキスト（補助） |
| `--text-light` | `#7a8e87` | 補助テキスト |

### 5.3 タイポグラフィ

| 用途 | フォント | ウェイト |
|------|----------|---------|
| 英字見出し・ロゴ | Cormorant Garamond | 300, 400, 500 |
| 日本語見出し | Noto Serif JP | 300, 400, 500 |
| 本文・ナビ・フォーム | Noto Sans JP | 300, 400 |

### 5.4 ロゴ

テキストロゴを使用する。

- 英語表記: `Social Support Ohana`（Cormorant Garamond, 300, letter-spacing: 0.12em）
- 日本語表記: `ソーシャルサポート おはな`（font-size: 12px, letter-spacing: 0.15em）

### 5.5 ヒーロー画像

- フリー素材を使用する
- テーマ: 自然、温もり、手を差し伸べるイメージ（福祉に合う柔らかい雰囲気）
- 形式: WebP推奨、1920x1280px以内

### 5.6 レスポンシブ対応

| デバイス | ブレークポイント |
|---------|----------------|
| モバイル | 〜768px |
| タブレット | 769px〜1024px |
| デスクトップ | 1025px〜 |

- モバイル時: ハンバーガーメニュー、1カラムレイアウト
- デスクトップ時: インラインナビ、2〜3カラムグリッド

### 5.7 アニメーション

- ヒーロー: フェードアップ（順次表示、0.2s間隔）
- スクロール: IntersectionObserverによるフェードイン（サービスカード、プロフィール等）

| スムーズスクロール | ページ内リンク時のスムーズスクロール |

### 6.3 スクロールアニメーション

- IntersectionObserverベースのフェードインアニメーション
